## Featured Projects

### [f1-insightx](https://github.com/DhruvShah-Dev/f1-insightx)
**Formula 1 analytics web app built around a race strategy simulator and fantasy lineup optimizer.**

- Combines a polished product surface with a backend/data foundation, rather than treating analytics as a standalone experiment.
- Demonstrates full-stack thinking across web delivery, server routes, and Python-based data ingestion and normalization.
- Shows how domain data can be turned into decision-support tooling with clear user-facing workflows.
- Reflects an engineering style that values usable analytics, structured systems, and product-minded implementation.

### [reddit-moderator-bot-detection](https://github.com/DhruvShah-Dev/reddit-moderator-bot-detection)
**Batch-based Reddit moderator analysis pipeline with explainable heuristic bot scoring and a Streamlit dashboard.**

- Built as a scalable, append-safe analysis pipeline for collecting moderator data across large subreddit batches.
- Emphasizes transparent, auditable scoring over black-box classification, making the system easier to inspect and reason about.
- Demonstrates pipeline design, resumable data collection, behavioral signal engineering, and investigation-oriented analytics.
- Treats data work as an operational system rather than a one-off script.

### [doc-ai](https://github.com/DhruvShah-Dev/doc-ai)
**Local, private document Q&A assistant for PDFs, DOCX, and TXT files.**

- Focuses on retrieval-oriented AI in a privacy-conscious setting without depending on cloud-hosted APIs.
- Demonstrates backend service design, document processing, vector search, and lightweight LLM integration.
- Turns document ingestion and search into a usable Q&A workflow instead of a purely technical demo.
- Shows practical AI applied where it is concrete and useful: querying personal or internal documents efficiently.
